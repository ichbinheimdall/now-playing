## About
A minimal static page that embeds kittinan's Spotify profile card. No backend, no API keys—just HTML, CSS, and a tiny script for auto-refresh.

## Usage
```bash
git clone https://github.com/ichbinheimdall/what-am-i-listen-rn
open index.html
```

Deploy anywhere: GitHub Pages, Vercel, Netlify, etc.

## Customize
- Change `uid` in the image URL to your Spotify user ID.
- Adjust refresh interval in `script.js`.

```
index.html   – single-file app
```
