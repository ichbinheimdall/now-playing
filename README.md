## About
A single-file static page showing my current Spotify track. No backend, no API keys — just one HTML file.

## Usage
```bash
git clone https://github.com/ichbinheimdall/now-playing
open index.html
```

Deploy anywhere: GitHub Pages, Vercel, Netlify, etc.

## Customize
- Change `uid` in the image URL to your Spotify user ID.
- Adjust refresh interval in `script.js`.

index.html   – single-file app
